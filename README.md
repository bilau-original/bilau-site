# 🍆 Bilau Clicker

Um jogo incremental clicker inspirado em Cookie Clicker.

## Jogar

Acesse [bilau.com.br](https://bilau.com.br) pra jogar!

## Funcionalidades

- **Clique pra crescer** — cada clique adiciona cm ao seu bilau
- **16 Construções** — de Mãozinha Amiga até Ocioverso, gerando cm/s passivo
- **60+ Melhorias** — poder de clique, multiplicadores, turbos por construção
- **10 Estágios de Evolução** — transformações visuais de 🌱 O Broto até ✴️ O Onibilau
- **Gotas Douradas** 💧 — eventos bônus aleatórios (Frenesi, Frenesi de Clique, Sorte, Corrente)
- **Sistema de Ascensão** — resete para ganhar Chips de Testosterona com bônus permanentes
- **10 Melhorias Celestiais** — bônus persistentes entre ascensões
- **75+ Conquistas** — marcos que concedem +2% CPS cada
- **Auto-save** — o jogo salva a cada 30 segundos com suporte a importar/exportar
- **Medidas astronômicas** — cm → m → km → notação científica

## Tecnologia

HTML/CSS/JS puro — sem build, sem dependências. Só abrir `index.html`.

# 🍆 Penis Clicker

An incremental clicker game inspired by Cookie Clicker.

## Play

Visit [bilau.com.br](https://bilau.com.br) to play!

## Features

- **Click to grow** — each click adds cm to your length
- **16 Buildings** — from Helping Hand to Idleverse, generating passive cm/s
- **60+ Upgrades** — click power, multipliers, building boosts
- **10 Evolution Stages** — visual transformations from 🌱 Sprout to ✴️ Omnipenis
- **Golden Drops** 💧 — random bonus events (Frenzy, Click Frenzy, Lucky, Chain)
- **Ascension System** — reset to earn Testosterone Chips for permanent boosts
- **10 Heavenly Upgrades** — persistent bonuses across ascensions
- **75+ Achievements** — milestones that grant +2% CPS each
- **Auto-save** — game saves every 30 seconds with import/export support
- **Astronomical measurements** — cm → m → km → scientific notation

## Tech

Pure HTML/CSS/JS — no build step, no dependencies. Just open `index.html`.
